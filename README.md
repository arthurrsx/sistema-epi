# Sistema de Gerenciamento de EPI

Projeto desenvolvido para controle de empréstimos de Equipamentos de Proteção Individual (EPI).

## Tecnologias utilizadas

- Python
- Django
- MySQL
- Bootstrap
- GitHub

## Funcionalidades

- Cadastro de colaboradores
- Listagem de colaboradores
- Edição de colaboradores
- Exclusão de colaboradores
- Pesquisa por nome

## Como executar o projeto

### Clonar repositório

git clone https://github.com/arthurrsx/sistema-epi.git

### Criar ambiente virtual

python -m venv venv

### Ativar ambiente virtual

venv\Scripts\activate

### Instalar dependências

pip install django pymysql

### Executar servidor

python manage.py runserver
